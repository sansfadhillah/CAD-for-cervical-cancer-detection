Mempermudah tugas Ahli Sitopatologi

NoMoreCans adalah aplikasi yang mempermudah tugas ahli Sitopatologi dalam proses screening pap smear test dengan memprediksi kelas citra pap smear menggunakan metode deep learning.

Tentang NoMoreCans

Aplikasi Berbasis Website Untuk Ahli Sitopatologi

Rumusan Masalah

menerapkan pendekatan deep learning menggunakan model convolutional neural network dalam memprediksi kanker pada citra pap smear dan mendapatkan model convolutional neural dengan performa terbaik

Deep Learning

Model deep learning yang digunakan Convolutional Neutral Network serta mendapatkan tingkat akurasi tertinggi sebesar 99.2% untuk asitektur AlexNet

Stack

Beberapa teknologi dibalik aplikasi NoMoreCans diantaranya, yaitu: Bootstrap 5.2, React, Django, REST API, dan Keras.
